Cascadia assets are secured by Safe. Safe is a popular and versatile multisignature wallet that provides advanced transaction capabilities and secure storage for digital assets. It meets best practice industry security standards (Formal Verification) and requires a predefined number of users to approve transactions. This prevents unauthorized access and unnecessary losses. Additionally, Safe allows for decentralized gasless voting, which makes governance more inclusive. The Cascadia Safe has been customized to meet the specific needs of the Cascadia network.

Deployed services include:

  • safe-client-gateway, which provides backend services for clients, such as transaction services and nodes;

  • safe-config-service, which provides configuration information in the context of the Safe client environment;

  • safe-transaction-service, which tracks Safe contract transactions; and

  • safe-wallet-web, which enables user asset management.
